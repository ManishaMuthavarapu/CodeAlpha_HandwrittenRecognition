# Handwritten Character Recognition using CNN

## 📌 Project Overview

This project recognizes handwritten digits using Deep Learning and Convolutional Neural Networks (CNN).

The model is trained using the MNIST dataset, which contains thousands of handwritten digit images.

---

## 🎯 Objective

To develop a deep learning model capable of recognizing handwritten digits accurately.

---

## 🛠 Technologies Used

* Python
* TensorFlow
* Keras
* NumPy
* Matplotlib
* OpenCV

---

## 📂 Dataset

MNIST Handwritten Digits Dataset

Dataset contains:

* 60,000 training images
* 10,000 testing images
* Digits from 0 to 9

---

## 🤖 Deep Learning Model

Convolutional Neural Network (CNN)

Layers Used:

* Conv2D
* MaxPooling2D
* Flatten
* Dense

---

## 📊 Model Accuracy

| Metric            | Score |
| ----------------- | ----- |
| Training Accuracy | 99%   |
| Testing Accuracy  | 98%   |
| Loss              | 0.05  |

---

## 📈 Features

* Handwritten Digit Recognition
* CNN-based Image Classification
* Deep Learning Model Training
* Prediction System
* Accuracy Evaluation

---

## 🧠 Conclusion

The Handwritten Character Recognition model successfully identifies handwritten digits with high accuracy using CNN architecture.

This project demonstrates the effectiveness of Deep Learning in image classification and character recognition tasks.
